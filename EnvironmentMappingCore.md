# EnvironmentMappingCore · Пространственная карта окружения KITI

## 1. Назначение

EnvironmentMappingCore отвечает за построение внутренней карты мира,
которую организм KITI использует для навигации, избегания препятствий,
планирования траекторий и предсказания опасных событий.

Модуль собирает данные со всех сенсоров и формирует единую модель среды,
которая передаётся в BrainCore, MotorPlanningCore и ReflexCore.

---

## 2. Источники данных

EnvironmentMappingCore получает сигналы от:

- SensorCore (видео, расстояние, касание, давление, звук);
- MotionPlanningCore (запросы на построение пространственных сеток);
- OrientationCore (положение тела и развороты);
- ReflexCore (случайные быстрые реакции на угрозы).

---

## 3. Основные функции

- построение 2D/3D карты окружающего пространства;
- разметка препятствий, стен, подвижных объектов;
- определение безопасных и опасных зон;
- прогнозирование траекторий объектов;
- оценка рисков столкновения;
- обновление карты в реальном времени.

---

## 4. Выходные данные

EnvironmentMappingCore передает:

- MotorPlanningCore → карту для построения маршрутов;
- ReflexCore → сигналы об угрозах;
- BrainCore → общую модель мира, как часть когнитивного слоя;
- StabilityCore → данные о поверхности под ногами;
- BodyCore → информацию о наклонах/перепадах.

---

## 5. Значимость в архитектуре

КИТИ не может двигаться, ориентироваться или выживать без карты окружения.

EnvironmentMappingCore делает поведение организма:

- предсказуемым,
- адаптивным,
- безопасным,
- логичным.

Это один из фундаментальных модулей автономного поведения.
