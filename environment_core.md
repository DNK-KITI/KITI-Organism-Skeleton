# ENVIRONMENT CORE · Модуль окружения KITI

Environment Core — это слой, который описывает мир вокруг организма.
Он даёт структурированное представление о грунте, наклоне, материалах,
температуре, рисках, препятствиях и динамике пространства.

Этот модуль НЕ управляет движением — он только даёт факты о среде.

---

## 1. Основные задачи

- анализ окружения,
- описание поверхности,
- фиксация опасностей,
- совместимость с StabilityCore,
- выдача данных в Motor Planning и Behavior Core,
- формирование SRL-тегов среды.

---

## 2. Модель поверхности

Каждый участок описывается параметрами:

- **тип покрытия**: твёрдое, мягкое, рыхлое, металл, бетон, лед, трава, жидкость;
- **сцепление (μ)**: коэффициент трения;
- **устойчивость**: есть ли провалы, неровности, пустоты;
- **жёсткость**: воспринимаемая нагрузка;
- **влажность**: влияет на скольжение;
- **температура**: влияет на сенсоры и моторы.

---

## 3. Обнаружение препятствий

Модуль фиксирует:

- выступы,
- бордюры,
- ступени,
- низкие препятствия,
- движущиеся объекты,
- опасные зоны (огонь, лёд, химия).

---

## 4. Интеграция с организмом

Environment Core передает данные в:

- **StabilityCore** — оценка риска падения;
- **Motor Planning Core** — корректировка шагов и траекторий;
- **Behavior Core** — выбор поведения;
- **SensorCore** — объединение сырых данных;
- **TerrainCore** — создание карты поверхности.

---

---

## 6. Назначение

Модуль является обязательным для:

- полноценной ходьбы,
- регистрации контактов с поверхностью,
- корректной работы баланса,
- адаптации движений под реальный мир.
