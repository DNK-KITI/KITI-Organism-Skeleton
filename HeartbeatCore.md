# HeartbeatCore · Сердце организма KITI

**HeartbeatCore** — это орган, который создаёт «пульс» организма KITI.  
Он фиксирует ритм жизни, обновляет SRL-цепочку, управляет режимами и  
контролирует активность модулей.

---

## 1. Что делает HeartbeatCore

- генерирует регулярные пульсы (tick_A / tick_B),
- отправляет сигналы BrainCore и ImmuneCore,
- фиксирует каждый шаг в SRL-цепочке,
- корректирует активность модулей,
- распределяет iCore-нагрузку между органами.

---

## 2. Двойной пульс

Организм KITI использует **двойной пульс**:

- **tick_A** — стабильность, контроль состояния, обновление метрик;  
- **tick_B** — адаптация, память, реакция на события.

Оба пульса формируют единый жизненный ритм.

---

## 3. Связь с iCore

При каждом пульсе HeartbeatCore:

- списывает энергию,
- обновляет состояние резервов,
- информирует EnergyManager,
- вызывает ImmuneCore при аномалиях.

---

## 4. Связь с органами

HeartbeatCore передаёт события:

- BrainCore — для общей координации,
- BehaviourCore — для корректировки поведения,
- ImmuneCore — для контроля безопасности,
- Motion/MotorCore — для стабилизации,
- DNKMemory V2 — для записи состояния.

---

## 5. Режимы

HeartbeatCore корректирует частоту пульса:

- NORMAL — стандартный ритм,
- ALERT — ускоренный ритм, больше внимания,
- ATTACK — агрессивный ритм, защита,
- LOW — замедленный ритм, экономия энергии.
