## 2. Основні модулі та функції

| Модуль | Функція |
|--------|----------|
| **Core / Heartbeat Engine** | керування циклом життя, ведення пульсу та журналу подій |
| **DNK-Vault / VAR** | шифроване зберігання журналів і станів |
| **Kupol / Watchdog** | самозахист, ізоляція та автоматичне відновлення |
| **Dragon / Snake / Tunnel** | багаторівнева система захисту каналів |
| **MINT-72 / 276** | цифрова ідентичність, автентифікація, тривожне слово |
| **Energy Manager** | керування живленням і режимами роботи |
| **Voice / CZAR-layer** | офлайн-взаємодія з людиною |
| **StopClicker** | апаратно-програмна кнопка аварійного зупинення |

---

## 3. Режими роботи

- **Offline-mode:** повна автономність, журнал записується на носій.  
- **Hybrid-mode:** короткочасна синхронізація через токен або QR-доступ.  
- **Kupol-mode:** автоматична ізоляція при аномаліях.  
- **Heartbeat-only:** мінімальне споживання енергії, лише мітки часу й стану.

## 4. Тести та результати

| Тест | Умови | Результат |
|------|--------|-----------|
| **Offline Integrity Proof** | 24 години роботи без мережі | лог цілісний, heartbeat без перерв |
| **Kupol Isolation** | примусове відключення живлення / мережі | автоматичне відновлення після рестарту |
| **Duress Test** | введення тривожного слова | збереження мінімального журналу, видалення ключів |
| **Energy Drain** | робота від powerbank до повного розряду | перехід у режим пульсу, журнал збережено |
| **Vault Proof** | верифікація hash-ланцюга у DNK-Vault | 100 % співпадіння контрольних хешів |
