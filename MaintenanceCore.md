# MaintenanceCore — модуль технического обслуживания организма

MaintenanceCore отвечает за внутреннюю самодиагностику, выявление отклонений, ведение журналов и планирование корректирующих действий.  
Это автономный блок поддержки жизненного цикла организма KITI.

Назначение:
1. Контроль состояния всех основных модулей.  
2. Обнаружение неисправностей и деградации компонентов.  
3. Поддержка периодических проверок.  
4. Генерация предупреждений для LogicCore и StabilityCore.  
5. Формирование планов восстановления.

Получает информацию от:
- MotorCore  
- SensorFusionCore  
- EnergyFlowCore  
- EnvironmentalCore  
- LogicCore  

Функции:
- диагностика модулей;  
- анализ аномалий;  
- фиксация внутренних событий;  
- подготовка рекомендаций для корректирующих действий.

Вход:
diagnostic_state

Выход:
maintenance_actions, warnings
