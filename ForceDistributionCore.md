# ForceDistributionCore — модуль распределения усилия

ForceDistributionCore управляет тем, как организм KITI распределяет физическое усилие между узлами, суставами и опорными точками.  
Задача модуля — сделать движение эффективным, безопасным и ресурсно устойчивым.

Назначение:
1. Распределение нагрузки между опорными элементами.  
2. Поддержка равновесия при смещении центра тяжести.  
3. Контроль предельных значений нагрузки.  
4. Выбор оптимальной схемы усилий при движении или удержании позиции.  
5. Предотвращение перегрузки отдельных узлов.

Использует данные:
- StabilityCore  
- MotorCore  
- BalanceAssistCore  
- LoadBalanceCore  
- SensorFusionCore  

Основные функции:
- расчет требуемой силы для каждой опорной точки;  
- перераспределение усилия при смене позы;  
- компенсация внешних нагрузок;  
- выбор энергоэффективного режима движения.

Вход:
body_state, stability_state, fused_frame

Выход:
force_map, correction_commands
