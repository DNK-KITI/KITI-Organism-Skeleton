# LoadBalanceCore · Распределение нагрузки KITI

## 1. Назначение модуля

LoadBalanceCore распределяет нагрузку по конечностям, корпусу, суставам
и точкам опоры организма. Модуль обеспечивает безопасное движение,
устойчивость при переносе веса, подъём грузов, а также защиту от перегрузок.

Основные функции:
- анализ текущей нагрузки на каждую конечность,
- мониторинг вертикальных и горизонтальных реактивных сил,
- предотвращение локальных перегрузок суставов,
- оптимизация распределения веса при движении,
- помощь MotorCore и BalanceCore при сложных манёврах,
- поддержка ImmuneCore в диагностике микроповреждений.

---

## 2. Источники данных

LoadBalanceCore получает данные от:

- SensorCore (датчики давления, контактные точки),
- BodyCore (положение конечностей),
- MotorCore (силовые параметры движения),
- BalanceCore (общий центр масс),
- StabilityCore (границы безопасных усилий).

---

## 3. Алгоритмы

- динамическое распределение вертикальной нагрузки,
- вычисление оптимального усилия для каждого сустава,
- анализ перегрузок на основе порогов ImmuneCore,
- компенсация нагрузки при внезапных изменениях опоры,
- адаптация движений для экономии энергии.

---

## 4. Выходные данные

LoadBalanceCore отдаёт:
- корректирующие усилия для MotorCore,
- предупреждения ImmuneCore о рисках повреждений,
- стабилизационные параметры для BalanceCore,
- отчёт о нагрузке для StabilityCore.

---

## 5. Взаимодействие в системе

LoadBalanceCore — критичный слой для реального робота KITI.

Он образует цепочку:
SensorCore → LoadBalanceCore → MotorCore / BalanceCore / ImmuneCore.

Без него организм теряет устойчивость, начинает повреждать суставы
и не способен безопасно взаимодействовать с физическим миром.
