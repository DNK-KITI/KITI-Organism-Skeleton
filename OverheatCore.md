# OverheatCore

## Назначение
Модуль контроля перегрева. Обнаруживает тепловые аномалии, анализирует риски повреждений и включает защитные меры для предотвращения выхода организма из строя.

## Функции
- мониторинг температуры по зонам;
- сравнение текущих значений с безопасными диапазонами;
- предупреждение о перегреве структурных модулей;
- передача команд в MotorSafetyLayer и ThermalCore для снижения нагрузки.

## Логика работы
Модуль постоянно анализирует входящие сигналы от ThermalCore и SensorCore.  
При превышении порогов активирует охлаждающие сценарии:  
уменьшение мощности, изменение движения, переход в безопасный режим, пауза нагрузки.

## Использование в организме
- предотвращение тепловых повреждений;
- поддержание работоспособности при высокой нагрузке;
- автоматическая адаптация поведения под тепловые условия.
