# LocalizationCore — модуль определения местоположения организма

LocalizationCore определяет местоположение цифрового организма KITI в физическом или виртуальном пространстве.  
Модуль обеспечивает точную локализацию для движения, навигации, взаимодействия и безопасности.

Назначение:
1. Определение позиции организма в координатах среды.  
2. Сопоставление локальной карты с фактическими данными.  
3. Поддержка автономной навигации и перемещения.  
4. Контроль ошибок позиционирования.  
5. Передача данных в OrientationCore и MovementCore.

Содержит:
- координатную систему;  
- карту окружения;  
- алгоритмы локализации (описание без реализации);  
- правила корректировки позиции.

Функции:
- оценка позиции по данным сенсоров;  
- сравнение фактического положения с картой;  
- корректировка локализации при расхождениях;  
- фиксация логов перемещения;  
- передача данных навигации.

Вход:
sensor_position_data, map_state

Выход:
localization_state, position_update
