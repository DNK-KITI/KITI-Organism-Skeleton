# PowerCore — модуль энергетической архитектуры организма

PowerCore описывает базовый энергетический контур организма KITI.  
Этот модуль определяет источники питания, зоны распределения энергии, приоритеты потребления и защитные ограничения.

Назначение:
1. Управление основной энергетической системой.  
2. Определение доступных источников питания (USB, аккумулятор, внешние линии).  
3. Передача параметров заряда и напряжения в EnergyManager.  
4. Формирование профилей энергопотребления для всех органов.  
5. Поддержка аварийных режимов и ограничений.

Содержит:
- параметры источников питания;  
- схемы энергетических цепей;  
- пороги напряжений;  
- минимальные уровни для жизни организма;  
- режимы ECO, NORMAL, HARD.

Функции:
- контроль поступления энергии;  
- распределение мощности;  
- управление переключением источников;  
- фиксация состояния энергетической линии;  
- работа под сменой нагрузки.

Вход:
power_state_request

Выход:
power_status, energy_limits
