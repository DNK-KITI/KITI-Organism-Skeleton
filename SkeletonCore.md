# SkeletonCore — базовый скелет цифрового организма

SkeletonCore определяет конструктивную основу организма KITI.  
Этот модуль описывает структуру тела, опорную систему и связи между узлами, на которые опираются органы и процессы организма.

Назначение:
1. Формирование общей архитектурной рамы организма.  
2. Определение точек крепления модулей и органов.  
3. Поддержка структурной целостности при изменениях конфигурации.  
4. Обеспечение стабильной геометрии взаимодействия органов.  
5. Передача статуса о деформациях и нагрузках.

Содержит:
- структуру базовых узлов;  
- таблицу связей между узлами;  
- описание уровней жёсткости;  
- правила распределения нагрузки;  
- структуру виртуального каркаса.

Функции:
- фиксация положения органов;  
- анализ структурных отклонений;  
- стабилизация каркаса при перегрузках;  
- передача параметров нагрузки в StabilityCore;  
- поддержка форм-фактора робота.

Вход:
structure_request

Выход:
structure_status, load_map
