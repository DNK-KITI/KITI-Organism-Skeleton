# LOGIC · Центральная логика поведения организма

Logic Layer — это уровень принятия решений организма KITI.
Он объединяет восприятие, состояние тела, цели и внутренние правила.

---

## 1. Основные функции Logic

- анализ входящих сигналов от Perception;
- оценка состояния тела (энергия, повреждения, перегрузки);
- применение Behaviour-политик;
- выбор допустимого действия;
- передача решений в Motor Planning / Reflex / Security.

Logic не управляет моторикой напрямую — он принимает решение,
а исполняющие модули делают действие.

---

## 2. Структура Logic Layer

### • State Analyzer  
Определяет:
- текущее состояние,
- уровень риска,
- допустимость действий.

### • Behaviour Selector  
Выбирает паттерн поведения:
- исследование,
- стабилизация,
- избегание,
- экономия энергии,
- защитный режим.

### • Decision Gateway  
Передаёт выбранное действие дальше по цепочке:
Motor Planning → Motor Control → тело.

---

## 3. Логирование решений

Каждое решение может записываться в SRL-трек:
- действие,
- причина,
- контекст,
- датчики,
- состояние организма.

---

## 4. Назначение Logic Layer

Logic — мозг поведения.
Он делает организм последовательным, адекватным и контролируемым.
